# Smart Campus System

**Smart Campus System** is a full-stack web application designed to modernize campus management by providing streamlined workflows for students, teachers, and admins. The project demonstrates real-time profile management, assignment handling, and admin functionalities, making it a solid showcase for SDE roles.

## Features

### Student
- View personal profile and role-specific information.  
- Access all assignments posted by teachers.  
- Track submission status and deadlines.  

### Teacher
- Create and manage assignments.  
- Track student submissions (mock implementation).  
- Quick dashboard with stats and recent actions.  

### Admin
- Manage users (students and teachers) and roles.  
- View system activity and statistics.  
- Approve or monitor registration requests.  

---

## Tech Stack

- **Frontend:** React.js, Redux Toolkit, TailwindCSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Atlas)  
- **Authentication:** JWT, Google OAuth  
- **Deployment:** Local / Cloud-ready  
- **API Client:** Axios
